# 인증 및 관리자 기능 구현 완료

## 📋 구현 개요

사용자 요청에 따라 **로그인 기능**과 **관리자 패널**을 완성했습니다.

---

## 🔐 1. 로그인 기능

### 📍 위치: `/login` 페이지

### 기능:
- **이메일 + 비밀번호 로그인**
  - 이메일 주소 입력
  - 비밀번호 입력
  - 로그인 버튼

- **인증 상태 관리**
  - Zustand를 사용한 상태 관리
  - localStorage 자동 저장
  - 페이지 새로고침 후에도 로그인 상태 유지

- **라우트 보호**
  - 로그인하지 않은 사용자는 자동으로 `/login`으로 리다이렉트
  - 로그인 페이지는 누구나 접근 가능
  - AuthProvider가 모든 라우트 보호

### 데모 계정:
```
📧 이메일: abc@naver.com
🔑 비밀번호: abc1234

역할: 관리자 (모든 권한)
```

### UI:
- 전문적인 그래디언트 배경
- 브랜드 로고 표시
- 에러 메시지 피드백
- 로딩 상태 표시

---

## 👥 2. 관리자 패널

### 📍 위치: AppHeader 드롭다운 → "관리자 패널" 버튼

### 기능:

#### 2.1 사용자 관리 탭
```
사용자 목록 표시
├─ 사용자 이름
├─ 이메일 주소
├─ 권한 (관리자 / 사용자)
├─ 권한 변경 버튼 (강등/승격)
└─ 삭제 버튼
```

**사용자 관리 규칙:**
- 현재 로그인한 사용자는 삭제 불가
- 마지막 관리자는 삭제 불가
- 자신은 권한 변경 불가

#### 2.2 사용자 추가 탭
```
사용자 추가 폼
├─ 이름 입력 (필수)
├─ 이메일 주소 입력 (필수, 중복 방지)
├─ 비밀번호 입력 (필수)
├─ 권한 선택 (사용자 / 관리자)
└─ 추가 버튼
```

**유효성 검사:**
- 모든 필드 필수 입력
- 이메일 중복 확인
- 입력 실패 시 에러 메시지 표시

## 🔑 3. 권한 시스템

### 권한 종류:
```typescript
type UserRole = 'user' | 'admin';

1️⃣ 사용자 (user)
   ├─ 회의 참석 가능
   ├─ 안건 검토 가능
   ├─ 표결 참가 가능
   └─ 관리자 패널 접근 불가

2️⃣ 관리자 (admin)
   ├─ 모든 사용자 기능
   ├─ 새 사용자 추가 가능
   ├─ 사용자 권한 변경 가능
   ├─ 사용자 삭제 가능
   └─ 관리자 패널 접근 가능
```

### 관리자 패널 접근:
```
AppHeader (우측 상단)
  ↓
사용자 아바타 클릭
  ↓
드롭다운 메뉴
  ├─ 로그인 계정 정보 표시
  ├─ 관리자만: "관리자 패널" 버튼
  └─ "로그아웃" 버튼
```

## 🏗️ 기술 구조

### 상태 관리 (authStore.ts)
```typescript
interface AuthState {
  currentUser: User | null;
  users: User[];
  login(email, password): boolean;
  logout(): void;
  addUser(email, name, password, role): boolean;
  updateUserRole(userId, role): boolean;
  deleteUser(userId): boolean;
  isAdmin(): boolean;
  isLoggedIn(): boolean;
}
```

### 데이터 저장소:
```
localStorage 키:
├─ auth_users          # 모든 사용자 목록 (JSON 배열)
└─ current_user        # 현재 로그인한 사용자 (JSON 객체)
```

### 초기 설정:
- 관리자 계정 자동 생성: `abc@naver.com` / `abc1234`
- localStorage 없으면 자동 초기화
- 관리자가 없으면 자동으로 추가

## 🔄 사용자 플로우

### 로그인:
```
1. 사이트 접속
   ↓ (로그인 필요)
2. /login 페이지로 자동 리다이렉트
   ↓
3. 이메일 + 비밀번호 입력
   ↓
4. "로그인" 버튼 클릭
   ↓
5. 메인 페이지 (/) 접속 가능
```

### 관리자 권한으로 사용자 추가:
```
1. AppHeader 우측 상단 아바타 클릭
   ↓
2. "관리자 패널" 버튼 클릭 (관리자만 표시)
   ↓
3. "사용자 추가" 탭 선택
   ↓
4. 이름, 이메일, 비밀번호, 권한 입력
   ↓
5. "사용자 추가" 버튼 클릭
   ↓
6. 새 사용자로 로그인 가능
```

### 권한 관리:
```
1. 관리자 패널 → "사용자 관리" 탭
   ↓
2. 사용자 목록에서 대상 선택
   ↓
3. "강등" 또는 "승격" 버튼 클릭
   ↓
4. 권한 즉시 변경 (로그인되어 있으면 자동 반영)
```

### 로그아웃:
```
1. AppHeader 우측 상단 아바타 클릭
   ↓
2. "로그아웃" 버튼 클릭
   ↓
3. /login 페이지로 리다이렉트
```
